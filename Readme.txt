Your Time To Shine: Island Existence (Test Version)
Presented by Nankidai

This is a simulation game spinoff of "Your Turn To Die -Death Game By Majority-", created as a bonus.

It contains spoilers for the original game, so it's recommended you play up to the end of Chapter 3, Part One first!
https://vgperson.com/games/yourturntodie.htm

Currently, in this test version, you can only play as Mishima and Kai.

■ Controls
PC: Left Click = Confirm, Right Click = Cancel, Ctrl = Skip Text
Phone: Tap = Confirm, Two-Finger Tap = Cancel
